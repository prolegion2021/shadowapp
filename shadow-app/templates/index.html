Endpoint

The endpoint for the API is /shadowban/<username> where <username> is the Reddit username you want to check.

Parameters

The API accepts the following parameters:

  • username (required): The Reddit username you want to check.
  • full (optional): If set to 1, the API will return additional information about the user, including join date, post karma, comment karma, and whether or not they have a verified email.

Responses

The API will return one of the following responses:

  • If the user is shadowbanned: {"is_shadowbanned": true}
  • If the user is not shadowbanned: {"is_shadowbanned": false} (if full parameter is not set) or {"is_shadowbanned": false, "username": "example", "join_date": "2022-01-01 12:00:00", "post_karma": 1000, "comment_karma": 500, "verified_mail": true} (if full parameter is set to 1).
  • If the username is invalid: {"error": "Invalid username"}
  • If there is a server error: {"error": "Internal Server Error"}

Examples

cURL

To check if a user is shadowbanned using cURL:

curl -X GET http://localhost:5000/shadowban/example

To check if a user is shadowbanned and get additional information using cURL:

curl -X GET "http://localhost:5000/shadowban/example?full=1"
